Restaurant order application

Technologies: Java with Spring Boot, React, MySql

Web application for a small restaurant business. The goal of the app is to enable online orders with or without a user being registered. It also implements an admin center where the owners can receive orders from customers in real time, add, remove and change the menu, receive feedback from customers, upgrade or revoke admin privileges, view the best and least selling menu items, as well as the busiest day during a time period, and finally view a graphical representation of the business' performance during a specified period.

Application for storing and editing notes

Technologies: Java with Spring Boot, React Native with Expo, MySql, SQLite

Mobile application built with react native that allows a user create notes and store them in categories. The categories can be color coded for easy identification. Users can back up and restore notes from any device and can set up automatic back up. The notes are stored in a SQLite database on the user's device. However, authentication and backup data are stored on the server with MySql
